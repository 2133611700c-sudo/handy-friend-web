# Evidence Retention

## Commit to repository

- Compact markdown reports
- Status index updates
- Remediation packets for failures

## Upload as workflow artifacts

- Screenshots
- Browser traces
- Large raw logs
- Large JSON outputs

## Retention policy

- GitHub artifact retention: 30 days
- Keep last 14 successful summaries
- Keep failure remediation history

## Forbidden

- Full secrets
- Customer PII
- Raw customer message bodies
