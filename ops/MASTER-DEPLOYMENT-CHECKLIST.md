```
  ⚠️ TT-YYYYMMDD-01   (Thumbtack) - Week 2
```

### How They're Used
```
1. Customer contacts you
2. Ask: "How did you find us?"
3. Note their response
4. When logging in Supabase: Set channel = tracking code
5. System automatically tracks & reports by channel

Example:
  Customer: "Found you on Google"
  You note: "GA-LSA-TVMOUNT" (Google LSA, TV Mounting)
  In Supabase: leads.channel = "GA-LSA-TVMOUNT"
  Report shows: LSA is generating X leads at $Y cost
```
